# AnonChat

AnonChat is a full-stack web application that allows users to post, comment, and interact anonymously. Built with a Java Spring Boot backend and a modern React frontend, AnonChat provides a safe and simple platform for open discussions without revealing user identities.

## Features

- Anonymous user signup and login
- Create, view, and delete posts
- Comment on posts anonymously
- Like/react to posts and comments
- User profile management
- Responsive and modern UI
